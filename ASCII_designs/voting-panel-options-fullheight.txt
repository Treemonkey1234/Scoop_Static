DESIGN NOTES:
=============
• All panels now utilize the full vertical height of the post
• Upvote button positioned at top
• Score centered vertically 
• Downvote button positioned at bottom
• Options 1, 3, 4, 5: Contained panels that stretch full height
• Option 2: Circular elements distributed across height
• Option 6: Floating elements without container for maximum space usage

This spreads out the voting elements to make better use of the available space and creates better visual balance with the post content.
